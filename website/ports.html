Ports

OpenWBEM is known to compile and run on the following platforms:
  • Linux (Distros: Caldera, RedHat, SuSE, Debian & others) (Architectures: i386, IBM s390, MIPS)
  • UnixWare 7
  • OpenServer 5
  • Sun Solaris 8 & 9
  • Mac OS X (Only partially supported. You will have to create your own build system, but the code compiles and the cimom works fine.)


Please notify us if you know of an additional platform that will build and run OpenWBEM or if you would like to see OpenWBEM ported to a platform where it does not currently work.
